Badflix api documentation

Registering users

Users can be registered through the subdomain /api/register/username/password/genres/birthdate
This get request returns a token, or in the case of a user already being registered, it will respond with the string "null", this is because our client dev cannot do anything competently.
The birthdate must be entered in the form dd,mm,yyyy

Reseting tokens

Users can be registered through the subdomain /api/token/username/password
This get request returns a token, or in the case of a set of invalid login details being passed, it will respond with the string "null", this is because our client dev cannot do anything competently.

Requesting films (token)

Users can be registered through the subdomain /api/films/token/{token}
The token of the user should be used in place of {token}
The request will return a list of youtube links seperated by commas

Posting likes

Users can be registered through the subdomain /api/like/{token}/{film_name}
The token of the user should be used in place of {token}
The name of the film should be used in place of {film_name}
The request will return yay if it runs without error
